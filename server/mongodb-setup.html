🌿 中医处方系统 - 云端配置向导

📋 配置步骤

1

创建 MongoDB Atlas 账户

访问 MongoDB Atlas 官网，使用邮箱注册免费账户。

⚠️ 推荐使用 Google 账户注册，速度更快！
2

创建免费集群

注册完成后，按照提示创建一个 FREE 级别的集群（共享 RAM，512MB 存储空间）。

选择区域：新加坡 (Singapore) 或 东京 (Tokyo) - 距离中国较近，速度更快。

3

配置数据库访问

在 Security → Database Access 中添加用户：

  • Authentication Method: Password
  • Username: 自定义用户名（如：tcm-admin）
  • Password: 强密码（至少12位）
  • Role: Read and write to any database
4

配置网络访问

在 Security → Network Access 中添加 IP 白名单：

点击 Add IP Address，选择 Allow Access from Anywhere (0.0.0.0/0)

⚠️ 这允许从任何地方访问数据库，生产环境建议限制特定 IP。
5

获取连接字符串

点击 Connect → Connect your application

复制连接字符串，类似：

mongodb+srv://username:password@cluster.mongodb.net/tcm-prescription?retryWrites=true&w=majority

将 username 和 password 替换为第3步创建的用户名和密码。

🚀 下一步

1. 配置环境变量

在 server/ 目录下创建 .env 文件，粘贴上方的配置内容。

2. 安装依赖

打开终端，运行：

cd server
npm install

3. 启动服务器

开发模式：npm run dev

或生产模式：npm start

4. 部署到云端

推荐使用 Vercel 或 Railway 部署后端服务。
