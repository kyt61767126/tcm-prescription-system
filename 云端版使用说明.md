## 部署步骤

### 第一步：配置 Supabase 数据库

1. **登录 Supabase**：https://supabase.com
2. **进入项目**：点击您的项目
3. **打开 SQL Editor**：左侧菜单点击 SQL Editor
4. **运行数据库脚本**：
   - 打开文件 `supabase/schema.sql`
   - 全选内容（Ctrl+A）
   - 复制（Ctrl+C）
   - 粘贴到 Supabase 的 SQL Editor
   - 点击 **Run** 执行

5. **获取配置信息**：
   - 在 Supabase 左侧菜单点击 **Settings** → **API**
   - 复制以下两个值：
     - **Project URL**：格式为 `https://xxxxx.supabase.co`
     - **anon public**：以 `eyJ` 开头的长字符串

### 第二步：配置 Netlify 环境变量

1. **登录 Netlify**：https://netlify.com
2. **进入项目**：点击您的项目（好奇精灵-4b5891）
3. **打开环境变量设置**：
   - 左侧菜单点击 **Site settings**
   - 点击 **Environment variables**
4. **添加环境变量**：
   - 点击 **Add a variable**
   - 变量名：`NEXT_PUBLIC_SUPABASE_URL`
   - 值：您的 Supabase Project URL
   - 保存
   - 再次点击 **Add a variable**
   - 变量名：`NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - 值：您的 Supabase anon public key
   - 保存

### 第三步：重新部署

1. **触发重新部署**：
   - 在 Netlify 左侧菜单点击 **Deploys**
   - 点击 **Trigger deploy** → **Deploy site**
2. **等待部署完成**：约1-2分钟

## Git自动上传配置

### 第一步：配置Git（如果还没配置）

1. **打开命令提示符**：按 Win+R，输入 `cmd`，回车
2. **配置用户名和邮箱**：
   ```cmd
   cd c:\Users\61767\Documents\trae_projects\kyt-zy\tcm-prescription-system
   git config --global user.name "kyt61767126"
   git config --global user.email "您的邮箱地址"
   ```

### 第二步：提交并上传代码

每次修改代码后，按以下步骤上传：

1. **打开命令提示符**，进入项目文件夹：
   ```cmd
   cd c:\Users\61767\Documents\trae_projects\kyt-zy\tcm-prescription-system
   ```

2. **查看修改的文件**：
   ```cmd
   git status
   ```

3. **添加所有修改**：
   ```cmd
   git add .
   ```

4. **提交修改**（说明这次改了什么）：
   ```cmd
   git commit -m "完善云端用户系统功能"
   ```

5. **上传到 GitHub**：
   ```cmd
   git push origin main
   ```

6. **Netlify 自动部署**：
   - 代码上传到 GitHub 后，Netlify 会自动检测到更新
   - 约1-2分钟后，云端系统会自动更新

### 快捷操作（每次修改后）

创建一个简单的批处理文件，一键上传：

1. **在项目文件夹中创建文件**：`一键上传.bat`
2. **写入以下内容**：
   ```batch
   @echo off
   cd /d "c:\Users\61767\Documents\trae_projects\kyt-zy\tcm-prescription-system"
   echo 正在添加修改...
   git add .
   set /p message="请输入更新说明："
   git commit -m "%message%"
   echo 正在上传到 GitHub...
   git push origin main
   echo.
   echo 上传完成！Netlify 会自动部署。
   pause
   ```
3. **双击运行**：每次修改后双击运行即可

## 使用说明

### 🏥 管理员操作

#### 1. 登录
- 访问您的 Netlify 网址：https://curious-sprite-4b5891.netlify.app
- 用户名：`admin`
- 密码：`admin123`

#### 2. 添加新大夫
- 登录后点击顶部 **【用户管理】**
- 填写信息：
  - 用户名（如：wangdoctor）
  - 密码（如：doctor123）
  - 医生姓名（如：王医生）
- 点击 **添加用户**
- 将用户名和密码告知对应的大夫

#### 3. 删除用户
- 在用户列表中找到要删除的用户
- 点击 **删除** 按钮
- 确认删除后，该用户将无法再登录

#### 4. 修改自己的密码
- 点击顶部 **【修改密码】**
- 输入原密码、新密码、确认新密码
- 点击 **确认修改**

### 👨‍⚕️ 普通大夫操作

#### 1. 登录
- 访问您的云端网址
- 使用管理员分配的用户名和密码登录

#### 2. 修改密码
- 登录后点击 **【修改密码】**
- 输入原密码、新密码、确认新密码
- 确认修改

#### 3. 使用系统功能
- 正常使用药品库、开处方、验方等功能
- 所有数据自动保存到云端
