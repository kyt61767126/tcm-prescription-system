# 中医处方系统专业版

## 项目介绍

这是一个专业的中医处方管理系统，提供完整的处方开立、药品管理、验方管理和病历管理功能。

### 主要功能

- **处方管理**：快速开方，支持药品搜索和自动填充
- **药品管理**：内置常用中药材库，支持自定义添加
- **验方管理**：保存常用方剂，支持快速调用
- **病历管理**：完整的处方历史记录和检索功能
- **打印功能**：精美的处方打印格式
- **价格计算**：自动计算药费和总费用

## 系统特色

- 🚀 **无限制使用**：完全免费，没有功能限制
- 🎨 **精美界面**：绿色主题，中医文化风格设计
- 💾 **数据本地存储**：所有数据安全保存在本地
- ⌨️ **便捷操作**：支持简码快速输入，键盘操作
- 📋 **完整功能**：从开方到管理一站式解决方案

## 安装和使用

### 开发者运行

```bash
# 进入项目目录
cd TCM-Prescription-System

# 安装依赖
npm install

# 开发模式运行
npm start

# 打包Windows版本
npm run build:win
```

### 用户使用

1. 运行 `npm start` 启动程序
2. 在左侧输入患者信息
3. 在药品表格中通过简码或名称搜索药品
4. 填写剂量后系统自动计算价格
5. 右侧实时预览处方效果
6. 点击保存或打印完成操作

## 文件结构

```
TCM-Prescription-System/
├── main.js              # Electron主进程
├── index.html           # 主界面文件
├── package.json         # 项目配置
├── data/                # 数据存储目录（自动创建）
│   ├── medicines.json   # 药品数据
│   ├── formulas.json    # 验方数据
│   └── prescriptions.json # 历史处方
└── assets/              # 资源目录
```

## 快捷键

- `F1` - 帮助
- `F2` - 药品管理
- `F3` - 验方管理
- `F4` - 病历管理
- `Ctrl+N` - 新建处方
- `Ctrl+S` - 保存处方
- `Ctrl+P` - 打印处方
- `Enter` - 确认选择药品
- `Esc` - 关闭弹窗
